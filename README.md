# Berlin Sales Analysis Project

## Overview
This project showcases essential Business Data Analytics skills through an analysis of sales data focused on Berlin. It integrates multiple tools and techniques to demonstrate proficiency in:

1. **Excel Dashboard**: Interactive visualization of sales performance.
2. **SQL Query Report**: Structured analysis for customer segmentation and trends.
3. **Python Notebook**: Data cleaning, exploration, and visualization.
4. **Tableau/Power BI Visualization**: Insights on market trends using interactive dashboards.

---

## Project Components

### 1. Excel Dashboard
- File: `Berlin_Sales_Dashboard.xlsx`
- Features:
  - Monthly sales performance.
  - KPIs: Total revenue and profit margin.
  - Filters for regions, products, and time periods.
- Tools: Microsoft Excel.

### 2. SQL Query Report
- File: `berlin_sales_sql_report.sql`
- Features:
  - Total spending and purchase frequency by customer.
  - Monthly sales trends.
  - Top-performing product categories.
- Tools: PostgreSQL/MySQL/SQLite.

### 3. Python Notebook
- File: `berlin_sales_python_analysis.ipynb`
- Features:
  - Data cleaning: Handling duplicates and formatting.
  - Analysis: Sales by product category and monthly trends.
  - Visualizations: Bar charts and line plots.
- Tools: Python (pandas, matplotlib, seaborn).

### 4. Tableau/Power BI Visualization
- File: `Cleaned_Berlin_Sales_Data.csv`
- Features:
  - Geographical heatmaps.
  - Monthly sales trends.
  - Product category performance.
- Tools: Tableau Public or Power BI.

---

## Dataset
- **Raw Data**: `Berlin_Sales_Data.csv`
  - Includes sales transactions, regions, and product categories.
- **Cleaned Data**: `Cleaned_Berlin_Sales_Data.csv`
  - Prepared for visualization with Tableau/Power BI.

---

## How to Use

1. **Excel Dashboard**:
   - Open `Berlin_Sales_Dashboard.xlsx`.
   - Create pivot tables and charts for interactive dashboards.

2. **SQL Queries**:
   - Load `Berlin_Sales_Data.csv` into your database.
   - Run the queries in `berlin_sales_sql_report.sql`.

3. **Python Notebook**:
   - Run `berlin_sales_python_analysis.ipynb` in Jupyter Notebook.
   - Analyze data and generate visualizations.

4. **Tableau/Power BI**:
   - Import `Cleaned_Berlin_Sales_Data.csv`.
   - Design dashboards with insights.

---

## Key Insights
- **Top Performers**: Electronics dominate sales, followed by Clothing.
- **Customer Trends**: Frequent buyers contribute significantly to revenue.
- **Monthly Trends**: Sales peak in Q4, aligning with holiday shopping.
